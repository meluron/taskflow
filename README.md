# taskflow

taskflow is a minimal, modern task manager with built-in time tracking and productivity visualization, all running locally in your browser.

## Features
- **Tasks & Subtasks:** Create, edit, reorder, and complete tasks easily
- **Time Tracking:** Start/stop timers for subtasks with a floating timer
- **Reports:** View daily and weekly time charts
- **Offline Use:** All data stored locally, no account or server needed
